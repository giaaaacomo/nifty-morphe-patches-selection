This bundle is intentionally independent from the official patch source and
contains only the patches maintained in this repository.

## Install

[Add this source to Morphe](https://morphe.software/add-source?github=giaaaacomo/nifty-patches-selection),
or add this repository URL manually:

`https://github.com/giaaaacomo/nifty-patches-selection`

Official Morphe patches and this bundle can be installed at the same time.
Enable prerelease patches for this source in Morphe to test builds published
from the `dev` branch.

## Development

Feature branches are merged into `dev` for prerelease testing. Stable releases
are produced from `main`. Build metadata, patch lists, release notes, and bundle
assets are generated by GitHub Actions.

Build locally with:

```sh
./gradlew :patches:buildAndroid generatePatchesList --no-daemon
```
